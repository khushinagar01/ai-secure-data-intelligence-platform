# AI Secure Data Intelligence Platform 🔐

##  Overview
AI-powered platform to analyze PDFs and log files for:
- Text summarization
- Sensitive data detection
- Security risk analysis

##  Features
- 📄 Upload PDF / Log files
- 🧠 Auto text extraction
- 🔍 Detect sensitive data (emails, passwords, API keys)
- ⚠️ Risk scoring system
- 📊 Clean UI output

##  Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Libraries: pdf-parse, multer

## ▶️ How to Run
```bash
npm install
node app.js
